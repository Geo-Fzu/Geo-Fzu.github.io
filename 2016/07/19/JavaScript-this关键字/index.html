JavaScript this关键字

upon.ink
2016-07-19
tech
JavaScript, 概念

this是 Javascript 语言的一个关键字,它代表函数运行时，自动生成的一个内部对象，只能在函数内部使用。随着函数使用场合不同，this的值会发生变化。但有一个总的原则，那就是this指的是调用函数的那个对象。

下面分五种情况，详细讨论this的用法。

  1. 纯粹的函数调用：

    这是函数最通常的用法，属于全局性调用，因此this就代表全局对象 Global（浏览器中的 window）。

  2. 作为对象方法调用：

    函数还可以作为某个对象的方法调用，这时this就指这个上级对象,即调用该方法的对象。

    • 示例：

  3. 在构造函数中调用：

    构造函数，就是通过这个函数生成一个新对象 object。这时，this 就指这个新对象。

  4. 在 apply 和 call 中调用：

    Function.prototype.apply()是函数对象的一个方法，它的作用是改变函数的调用对象，它的第一个参数就表示改变后的调用这个函数的对象。

  5. 箭头函数中的 this：

    箭头函数是 ES6 中的新特性，函数体内的this对象，就是定义时所在的对象，而不是使用时所在的对象。
